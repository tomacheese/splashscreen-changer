# GEMINI.md

## 目的
このファイルは、Gemini CLI を使用して開発を行う際のコンテキストと作業方針を定義します。

## 出力スタイル
- **最終的な回答**: 日本語
- **途中経過**: 重要な点を除き英語
- **コード内のコメント**: 日本語
- **エラーメッセージ**: 英語

## 共通ルール
- **会話言語**: 日本語
- **コミットメッセージ**: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) に従う。`<description>` は日本語。
- **ブランチ名**: [Conventional Branch](https://conventional-branch.github.io) に従う。`<type>` は短縮形 (feat, fix)。
- **表記**: 日本語と英数字の間に半角スペースを挿入する。

## プロジェクト概要
- **目的**: EasyAntiCheat を使用しているアプリケーション（主に VRChat）の起動画面（スプラッシュスクリーン）をランダムに変更する。
- **主な機能**:
    - 起動ごとのランダム画像設定
    - 画像の自動処理（リサイズ・クロップ不要）
    - 設定ファイルによるカスタマイズ
    - 自動起動設定

## コーディング規約
- **言語**: Go
- **フォーマット**: `go fmt` に従う。
- **コメント**: 関数やインターフェースには docstring を日本語で記載する。
- **エラーメッセージ**: 先頭に絵文字がある場合は統一する。

## 開発コマンド

```bash
# 依存関係のインストール
go mod download

# テスト実行
go test -v ./...

# フォーマット確認
go fmt ./...
git diff --exit-code

# ビルド
go build ./cmd/splashscreen-changer
```

## 注意事項
- **セキュリティ**: 認証情報や機密情報をコードやログに含めない。
- **既存ルールの優先**: プロジェクトに既存のルールがある場合はそれに従う。
